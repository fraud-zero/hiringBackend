Introduction

This api documentation is for the hiring process of Fraud0

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Placement endpoints

Display a listing of placements.

Example request:
curl --request GET \
    --get "http://localhost/api/placements" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"sort_by\": \"total\",
    \"sort_order\": \"desc\",
    \"per_page\": 10,
    \"page\": 1,
    \"paginate\": true,
    \"key\": \"\\\"\\\"\",
    \"platform\": \"\\\"\\\"\",
    \"min_total\": 0,
    \"min_invalid_total\": 0
}"

const url = new URL(
    "http://localhost/api/placements"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "sort_by": "total",
    "sort_order": "desc",
    "per_page": 10,
    "page": 1,
    "paginate": true,
    "key": "\"\"",
    "platform": "\"\"",
    "min_total": 0,
    "min_invalid_total": 0
};

fetch(url, {
    method: "GET",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Request      

GET api/placements

Headers

Content-Type      

Example: application/json

Accept      

Example: application/json

Body Parameters

sort_by   string  optional  

Field to sort by. Possible values: key, platform, total, invalid_total, invalid_total_percent, pixel_stuffing, pixel_stuffing_percent, viewable, viewable_percent, non_viewable, non_viewable_percent, mfa_site_symptoms, mfa_site_symptoms_percent, other_invalid, other_invalid_percent. Example: total

sort_order   string  optional  

Sort order. Possible values: asc, desc. Example: desc

per_page   integer  optional  

Number of results per page. Example: 10

page   integer  optional  

Page number. Example: 1

paginate   boolean  optional  

Whether to paginate results. Default value: true. Example: true

key   string  optional  

Optional. Filter by key (case-insensitive). Example: ""

platform   string  optional  

Optional. Filter by platform. Possible values: ["","app", "web"]. Example: ""

min_total   integer  optional  

Optional. Filter placements with total greater than or equal to this value. Example: 0

min_invalid_total   integer  optional  

Optional. Filter placements with invalid_total greater than or equal to this value. Example: 0
